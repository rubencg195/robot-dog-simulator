# Robot Dog Simulator

App available in AI Studio: https://ai.studio/apps/1057c725-dec1-46c1-b526-fcaa2c034288

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
